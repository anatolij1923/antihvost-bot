# 📐 Архитектура бота `antihvost-bot`

Проект телеграм-бота, автоматизирующего учёт лабораторных, домашних заданий и рейтингов. Ниже представлена структура файлов и каталогов.

```text
antihvost-bot/
├── bot.py               # Точка входа: запуск бота
├── config.py            # Конфиги: токены, настройки БД
├── .env                 # Секреты (НЕ пушить в git!)
├── requirements.txt     # Зависимости проекта

├── handlers/            # Обработчики команд
│   ├── __init__.py
│   ├── start.py         # Команда /start, авторизация
│   ├── menu.py          # Главное меню
│   ├── labs.py          # Работа с лабораторными
│   ├── homework.py      # Работа с домашками
│   └── rating.py        # Работа с рейтингом

├── keyboards/           # Reply и Inline-клавиатуры
│   ├── __init__.py
│   ├── main_menu.py
│   ├── labs_menu.py
│   └── rating_menu.py

├── states/              # Состояния FSM
│   ├── __init__.py
│   ├── auth_states.py   # Авторизация
│   ├── lab_states.py    # Ввод лабораторных
│   └── homework_states.py

├── database/            # Работа с базой данных
│   ├── __init__.py
│   ├── db.py            # Подключение к БД
│   ├── models.py        # SQLAlchemy-модели (если используется)
│   └── queries.py       # Основные SQL-запросы

├── utils/               # Вспомогательные функции
│   ├── __init__.py
│   ├── validators.py    # Валидация ввода
│   ├── formatters.py    # Форматирование текста
│   └── logger.py        # Логирование и дебаг

└── data/                # Статический контент
    └── static/          # Мотивации, анекдоты, фикс. тексты
```
